We have a browser that runs on our personal system to display files found on the web.
Internet is worlds largest computer network.
HTTP is the main protocol of the web. Internet uses HTTP to communicate with us, the browser and servers.
And servers are the computers that host the files on the web.


HTML (Hyper Text Markup Language) files are main type of documents on the web and Hyperlinks are links between web pages.


HTML Tags:


1. Bold tag to bold the text.
2. EM or Italic tag to italicise the text.
3. HTML Attributes: Some of the tags consist of additional values called Attributes.
4. Anchor tag to make links.
5. Image (img) tag to add images to the web page.
6. Whitespace (br) tag for new line.
7. Paragraph (p) tag for making text block.
8. div and span are container tags.

Inline versus Block tags: Some tags are inline tags (br, a, img etc.) thats don't make any blocks but some are block tags (p, div, form etc.) that makes block with width and height.

HTML Document contains Docktype, head and body tags for partition of the webpage.
